## Preferences

### Change the browser:
Ensure that the desired browser is installed.
```BASH 
which google-chrome
which chromium
# and so on...
```
Change the Browser field in the preferences to the selected browser.
- xdg-open - **System Default**
- firefox - Firefox
- google-chrome - Google Chrome
- google-chrome-stable - Google Chrome
- chromium - Chromium
- opera - Opera
- msedge - Microsoft Edge

****EXTRA****
*On Google Chrome is possible open the browser in a chosen profile*
just use the option below.
google-chrome --profile-directory="Profile 1"
On the chrome folder to select the Chrome profile
Mac folder ~/Library/Application Support/Google/Chrome
Linux folder ~/.config/google-chrome 
Windows folder %USERPROFILE%\AppData\Local\Google\Chrome\User Data
